## 推荐学习路径

```
第一阶段（1-2周）：
  ↓
  安装 Anaconda + VSCode
  学习 pandas / numpy 基础
  用 yfinance 拉一只股票数据画K线
  ↓
第二阶段（2-4周）：
  ↓
  用 Backtesting.py 跑一个简单均线策略
  理解夏普比率、最大回撤
  ↓
第三阶段（1-2月）：
  ↓
  选定平台（vnpy / 聚宽 / 自建）
  对接券商 API 或 Paper Trading
  实现止损止盈逻辑
  ↓
第四阶段（持续）：
  ↓
  模拟盘验证3个月以上
  逐步小资金实盘
```
