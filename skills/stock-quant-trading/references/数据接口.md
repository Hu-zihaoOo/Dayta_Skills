# 数据接口

## A股数据

### Tushare
- **描述**: A股最全免费数据接口，覆盖基本面、行情、因子等
- **链接**: https://tushare.pro/
- **免费额度**: 基础数据免费，量化数据需积分
- **适合场景**: 入门首选

### 聚宽（JoinQuant）
- **描述**: 数据+回测+模拟盘一站式平台，数据质量高
- **链接**: https://www.joinquant.com/
- **适合场景**: 研究+模拟盘一体化用户

### 米筐（RiceQuant）
- **描述**: 类似聚宽，提供完整量化工具链
- **链接**: https://www.ricequant.com/

### AkShare
- **星标**: 活跃
- **语言**: Python
- **描述**: 覆盖股票、期货、外汇、数字货币的全面金融数据接口，免费
- **链接**: https://github.com/jindai234/AkShare
- **适合场景**: 一个库覆盖多市场数据

## 全球市场数据

### Yahoo Finance（yfinance）
- **语言**: Python
- **描述**: 免费获取美股/港股历史数据，极简 API
- **安装**: `pip install yfinance`
- **适合场景**: 美股/港股数据获取

### ccxt
- **星标**: 30k+ ⭐
- **描述**: 加密货币+股票交易所统一API封装，支持 Binance/OKX/Coinbase等，支持 WebSocket
- **链接**: https://github.com/ccxt/ccxt
- **适合场景**: 数字货币自动交易、多交易所切换

### Alpha Vantage
- **描述**: 免费美股 API，支持股票价格、财务数据、指数
- **链接**: https://www.alphavantage.co/
- **免费额度**: 每分钟5次，每日500次

### Financial Modeling Prep
- **描述**: 美股财务数据 API，免费额度较充足
- **链接**: https://site.financialmodelingprep.com/

## Level 2 / 高频数据（A股）

| 来源 | 说明 | 成本 |
|------|------|------|
| 券商柜台（XTP/USTP） | 最权威，直接从交易所拉取 | 较高 |
| 聚宽/米筐 | Level2数据包 | 付费 |
| 东方财富 Choice | 专业终端 | 付费 |
| 同花顺 iFinD | 专业终端 | 付费 |

> 超短线需要 Level2 数据（订单簿），免费渠道较少，建议先用1min K线起步。
