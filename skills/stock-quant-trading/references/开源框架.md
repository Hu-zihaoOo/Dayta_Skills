# 开源框架

## A股 / 国内通用

### vnpy
- **星标**: 19.1k ⭐（GitHub）
- **语言**: 纯 Python
- **协议**: MIT
- **描述**: 国内最活跃的量化交易平台框架，支持期货、期权、股票、ETF等多品种，提供 VN Station 图形界面。可对接国内主流券商（CTP、XTP等）。
- **链接**: https://github.com/vnpy/vnpy
- **推荐理由**: 生态最成熟，文档丰富，社区活跃；实盘首选

### pyalgotrade-cn
- **描述**: pyalgotrade 的中国A股适配版，支持国内券商对接
- **适合场景**: 策略回测，熟悉 Python 者快速入门

### EasyTrader
- **描述**: 基于 `pywinauto` 模拟同花顺客户端操作，实现自动化交易；散户友好，免费
- **适合场景**: 无券商API接口的散户，不想写太底层的东西
- **注意**: 依赖同花顺客户端前台运行，稳定性一般

### pytradecn
- **描述**: 券商客户端自动化操作库，采用客户端驱动型工厂模式，支持所有中国券商客户端
- **链接**: https://github.com/jadepeng/pytrader
- **适合场景**: 东方财富用户

### StockSharp（>S#）
- **星标**: 12k+ ⭐
- **语言**: C# 为主
- **描述**: 跨市场全品种（股票/期货/外汇/加密货币/期权），支持美股/A股/港股
- **链接**: https://github.com/StockSharp/StockSharp
- **适合场景**: 多市场玩家，有 C# 基础

## 全球通用 / Python 回测

### Backtesting.py
- **星标**: 4k+ ⭐
- **语言**: Python
- **描述**: 轻量级回测框架，API 简洁，支持止损/止盈/杠杆
- **链接**: https://kernc.github.io/backtesting.py/
- **适合场景**: 快速验证策略思路，无需复杂架构

### pybroker
- **星标**: 活跃开发中
- **语言**: Python
- **描述**: 支持机器学习的量化交易框架，内置 `stop_loss_pct`、`take_profit_limit` 等参数
- **链接**: https://github.com/edtechre/pybroker
- **适合场景**: 想用 ML 做 alpha 因子挖掘

### AutoTrader
- **描述**: 从回测到实盘一站式 Python 平台，支持自定义止亏/止盈、自定义指标
- **链接**: https://github.com/kieran-mackle/AutoTrader
- **适合场景**: 有一定经验，想一套代码覆盖回测+优化+实盘

### VectorBT
- **星标**: 7k+ ⭐
- **语言**: Python
- **描述**: 向量化回测，性能极快，支持参数扫描
- **适合场景**: 高频参数优化、策略寻优

### Zipline
- **星标**: 18k+ ⭐（by Quantopian）
- **语言**: Python
- **描述**: 学术/工业界最常用的回测框架之一，Quantopian 自用框架开源
- **适合场景**: 因子研究、学术量化策略

## 全球通用 / 实盘执行

### openalgo
- **星标**: 活跃
- **描述**: 开源算法交易平台，提供统一 API 层，支持 30+ 券商，覆盖 Indian brokers/NSE
- **链接**: https://github.com/marketcalls/openalgo
- **适合场景**: 多券商聚合，非A股用户参考
